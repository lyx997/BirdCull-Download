# BirdCull v1.0.1

## Downloads

- `BirdCull_Setup_win10_win11.exe`
- `BirdCull_Setup_win10_win11-1.bin`

Both files are part of the same installer set and should be distributed together.

## Highlights

- Single installer flow: start in Free Edition and unlock Full Edition after activation
- Built-in activation entry with local device code
- Free Edition limit: up to the first 200 photos per capture date
- Hard-link based output to avoid duplicating original photos
- Chinese and English interface support

## System

- Windows 10 / Windows 11
- NTFS recommended
- NVIDIA RTX series GPU recommended
- 16 GB RAM or more recommended
